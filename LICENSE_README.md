# 授权系统使用说明

## 系统概述

授权系统实现了以下功能：
- **使用次数限制**：
  - 首次使用：需要先解锁（0次）
  - 解锁后：30 次
- **导出成功后扣次数**：只有成功导出笔记后才消耗使用次数
- **隐蔽存储**：授权数据分散存储在系统目录中，无 wiz 字样
- **解锁码机制**：用户首次使用需联系你获取解锁码，解锁后获得30次使用次数

## 文件说明

| 文件 | 用途 | 分发方式 |
|------|------|---------|
| `license_manager.py` | 授权管理核心模块 | 随程序分发 |
| `key_generator.py` | 解锁码生成器（仅你使用） | **不要分发给用户** |
| `private_key.txt` | 私钥文件 | **不要分发给用户，不要提交到git** |

## 使用流程

### 首次使用（需要解锁）

**1. 用户首次运行程序**

程序显示：
```
============================================================
           ⚠️ 需要解锁
============================================================

本工具需要解锁后才能使用。

请联系我获取解锁码：

  QQ: 843115404

------------------------------------------------------------
您的机器码: 4F0C-DDCE-0E92-1852
------------------------------------------------------------

请复制以上机器码发送给我，我会为您生成解锁码。
============================================================

是否有解锁码? (y/n):
```

**2. 用户联系你**
- 用户发送机器码（如 `4F0C-DDCE-0E92-1852`）

**3. 你生成解锁码**

```bash
python key_generator.py <机器码>

# 示例
python key_generator.py 4F0C-DDCE-0E92-1852
```

输出：
```
============================================================
           解锁码生成成功
============================================================

机器码: 4F0C-DDCE-0E92-1852
解锁码: inEneEaZcWzZ4Ukkyt1oAw==

============================================================
请将此解锁码发送给用户
============================================================
```

**4. 用户输入解锁码**
- 程序验证解锁码
- 验证通过后，用户获得30次使用次数
- 用户现在可以正常使用工具导出笔记

### 后续使用

1. **导出笔记**：用户正常使用工具导出笔记
2. **扣减次数**：每次成功导出后，自动扣减1次
3. **次数显示**：导出完成后显示剩余次数
4. **次数用完**：需要再次联系你获取解锁码
5. **再次解锁**：每次解锁都会重置为30次

### 关键特点

- **启动不扣次数**：程序启动时不扣次数，只有导出成功后才扣
- **失败不扣次数**：导出失败、取消操作都不扣次数
- **重装仍受限**：删除程序重新安装，授权数据仍然存在，次数不会重置

## 技术细节

### 隐蔽存储位置

**macOS：**
- `~/.local/share/.fontconfig/.fonts.conf.d/.uuid`
- `~/.ssh/.known_hosts.old`
- `~/.config/pip/.cache.json`

**Windows：**
- `%USERPROFILE%/.vscode/extensions/.ext.db`
- `%USERPROFILE%/.docker/machine/machines/.meta`
- `%APPDATA%/Code/User/globalStorage/.state.vscdb`

### 解锁码原理

1. 程序基于硬件特征生成**机器指纹**
2. 机器码 = 机器指纹前16位（格式化显示）
3. 解锁码 = HMAC-SHA256(机器码, 私钥) 的前16字节Base64编码
4. 验证时重新计算并比较，确保只能由持有私钥的人生成有效解锁码

### 安全性说明

- **防重装**：授权数据存储在用户目录的隐蔽位置，删除程序不会删除授权数据
- **防伪造**：没有私钥无法生成有效解锁码
- **防篡改**：授权数据使用机器指纹加密，复制到其他机器无效
- **防绕过**：只有导出成功后才会扣次数，避免误扣

## 修改使用次数限制

如需修改使用次数，编辑 `license_manager.py`：

```python
class LicenseManager:
    # 使用次数限制
    FIRST_TIME_LIMIT = 0      # 首次使用限制（0 = 需要解锁）
    RESET_LIMIT = 30          # 解锁后限制
```

## 注意事项

1. **备份私钥**：`private_key.txt` 是生成解锁码的唯一凭证，请妥善保管
2. **不要提交**：确保 `private_key.txt` 在 `.gitignore` 中
3. **不要分发**：`key_generator.py` 和 `private_key.txt` 不要分发给用户
4. **虚拟机**：虚拟机环境硬件指纹可能变化，建议告知用户尽量在物理机上使用

## 测试授权系统

```bash
# 查看当前授权状态
python license_manager.py --status

# 使用解锁码解锁/重置
python license_manager.py --reset <解锁码>
```
